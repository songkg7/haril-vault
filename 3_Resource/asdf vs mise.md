## Conclusion

mise 승.

- asdf 의 기능을 제공해주면서도 더 좋은 사용자 경험 및 더 직관적인 명령어셋
- asdf 를 쓸 때는 명령어가 기억이 안나서 수시로 docs 를 뒤져봐야 했었는데, mise 를 사용한 이후로는 그런 경우가 많이 줄어듬
- 언어 버전 뿐만 아니라, 환경변수 기능도 꽤나 유용. direnv 를 대체할 수 있을듯
