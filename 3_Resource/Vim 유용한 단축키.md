```
<C-w>h : 화면분할 상태에서 좌측 창으로 커서 이동
<C-w>l : 화면분할 상태에서 왼쪽 창으로 커서 이동
<C-w>w : 다음 창으로 커서 이동
<C-w>W : 이전 창으로 커서 이동
<C-w>r : 다음 화면과 위치 바꾸기
<C-w>o : 현재 화면만 두고 나머지 분할 창 닫기
:qa(qall) : 모든 창 닫기
```
