디자인 패턴은 공통된 설계 문제들을 해결하기 위해 마련된 일종의 설계 원칙에 가깝다.

디자인 패턴은 크게 생성 패턴, 행위 패턴, 구조 패턴 이 세가지 종류로 구분된다.

#### 생성 패턴

- 추상 팩토리
- 빌더
- 팩토리 메서드
- 싱글톤

#### 행위 패턴

- 어댑터
- 컴포지트
- 데코레이터
- 파사드
- 프록시

#### 구조 패턴

- 책임 연쇄
- 커맨드
- 전략
- 옵서버
- 템플릿 메서드
