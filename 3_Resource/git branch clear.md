[[GitHub]] 의 CLI 인 `gh` 를 사용하면 로컬의 브랜치를 쉽게 정리할 수 있다.

```bash
brew install gh
```

`poi` 를 설치한다.

%% 설치 방법 정리 %%

```bash
gh poi
```
