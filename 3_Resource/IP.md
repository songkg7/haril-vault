# What is IP?

- 지정한 IP 주소에 데이터 전달
- 패킷(packet)이라는 통신 단위로 데이터 전달

1. 출발지 IP, 목적지 IP, message 를 적어서 패킷을 만듬
2. 인터넷 망으로 패킷을 던짐
3. 노드에서 확인을 거치며 목적지로 전달
