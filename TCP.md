> 전송 제어 프로토콜 (Transmission Control Protocol)

# IP stack 4계층

1. Link - 네트워크 인터페이스 계층
2. IP - 인터넷 계층
3. TCP, UDP - 전송 계층
4. HTTP, FTP - 애플리케이션 계층

## 특징

- 연결지향
- 데이터 전달 보증
- 순서 보장
- 신뢰할 수 있는 프로토콜
- 현재는 대부분 TCP 사용
