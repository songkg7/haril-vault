## What is interface?

[[Java]] 에서 interface 는 다음과 같은 class 를 표현합니다.

1. 인스턴스화 될 수 없다.
2. 모든 멤버 메서드는 public abstract 이기 때문에 sub class 에서의 구현을 강제한다.

좁은 의미에서만 본다면 이게 가장 정확한 설명일 것 입니다.

하지만 객체지향 관점에서 인터페이스는 **추상화** 라는 큰 임무를 가지고 있습니다. 그럼 우선 추상화가 뭔지 알아야 합니다.
